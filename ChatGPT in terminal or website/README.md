# ChatGPT in terminal or website

在终端或浏览器中与 AI 对话

# 安装步骤

```bash
pip -i https://pypi.doubanio.com/simple -r requirements.txt
```

在 api.json 写入 key,可更改使用模型

# 使用说明

1. 终端中使用

```bash
python main.py
```

> `json`中更改`model`的值以更改[使用模型](https://help.aliyun.com/zh/model-studio/getting-started/models)

2. 浏览器中使用

```bash
python web.py
```
