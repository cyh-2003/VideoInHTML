用于玩原神看攻略视频时控制小窗口的播放<br>
浏览器后要添加`--remote-debugging-port=9222`<br>
Chrome和Edge都可以
| 按鍵 | 操作         |
| ---- | ------------ |
| ←  | 快退 3 秒    |
| →  | 快进 3 秒    |
| r  | 播放或暂停    |

json格式
```json
{
    "数字": "中文+BV号"
}
```
示例
```json
{
    "1": "原海异种BV1yM4m1z7rS",
    "2": "海露花BV15E421N7T1"
}
```
